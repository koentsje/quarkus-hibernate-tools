== Installation

If you want to use this extension, you need to add the `io.quarkiverse.hibernate:quarkus-hibernate-tools` extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>io.quarkiverse.hibernate</groupId>
    <artifactId>quarkus-hibernate-tools</artifactId>
    <version>{project-version}</version>
</dependency>
----
